## Setup & Installation

### 1. Clone the repo
git clone https://github.com/your-username/alkira-mfa-login.git
cd alkira-mfa-login

### 2. Install Dependencies
cd backend && npm install
cd ../frontend && npm install

### 3. Run the backend
cd backend
node server.js

### 4. Run the frontend
cd frontend
npm run dev

Frontend runs at: http://localhost:5173
Backend runs at: http://localhost:8080
